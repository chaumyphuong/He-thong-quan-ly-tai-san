# HỆ THỐNG QUẢN LÝ TÀI SẢN (ASSET MANAGEMENT SYSTEM)
- Author: Chau My Phuong
- Date: 20/12/2022
---

## 1. Giới thiệu
Dự án *Hệ thống quản lý tài sản* được xây dựng nhằm hỗ trợ doanh nghiệp quản lý tài sản nội bộ như: theo dõi thông tin tài sản, phân loại, khấu hao và tình trạng sử dụng.

Project tập trung vào thiết kế và triển khai cơ sở dữ liệu bằng SQL.

---

## 2. Mục tiêu dự án
- Phân tích nghiệp vụ quản lý tài sản trong doanh nghiệp
- Thiết kế cơ sở dữ liệu đáp ứng yêu cầu nghiệp vụ
- Xây dựng SQL Script để tạo và quản lý dữ liệu
- Cung cấp dữ liệu mẫu phục vụ truy vấn và báo cáo

---

## 3. Phạm vi hệ thống
Hệ thống hỗ trợ các chức năng chính:
- Quản lý thông tin tài sản
- Phân loại tài sản
- Theo dõi thời gian mua và khấu hao
- Quản lý phòng ban / đơn vị sử dụng tài sản
- Truy vấn và báo cáo dữ liệu tài sản

---

## 4. Vai trò trong dự án
Trong dự án này, tôi đảm nhận vai trò:
- Phân tích yêu cầu nghiệp vụ quản lý tài sản
- Chuyển đổi yêu cầu nghiệp vụ thành mô hình dữ liệu
- Thiết kế các bảng, khóa chính, khóa ngoại
- Xây dựng các truy vấn SQL, View và Stored Procedure
- Chuẩn hóa dữ liệu và đảm bảo tính toàn vẹn

---

## 5. Thiết kế cơ sở dữ liệu
### Các thành phần chính:
- **Bảng dữ liệu**: tài sản, loại tài sản, phòng ban, nhân viên (tùy phạm vi)
- **Khóa chính / khóa ngoại** đảm bảo quan hệ dữ liệu
- **ER Diagram** mô tả mối quan hệ giữa các bảng

> Chi tiết cấu trúc bảng được mô tả trong thư mục thiết kế và script SQL.
